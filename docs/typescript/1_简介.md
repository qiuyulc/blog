# 简介

## 1.概述

TypeScript 是微软公司开发的一种基于 JavaScript 语言的变成语言。目的是为了增强 JavaScript 的功能，可以将它看成 JavaScript 的超集。

## 2.类型的概念

类型(type)指的是一组具有相同特征的值。如果两个值具有某种相同的特征，就可以说，他们属于同一种类型。

类型是人为添加的一种编程约束和用法提示。

## 3.动态类型与静态类型

JavaScript 属于动态类型，js 解析器可以根据赋值自动分析变量类型。

```js
let x = 1;
x = "hello";

let y = { foo: 1 };
delete y.foo;
y.bar = 2;
```

TypeScript 引入了一个更强大、更严格的类型系统，属于静态类型语言
上面的代码在 TypeScript 中都会报错。

## 4.静态类型的有点

1. 有利于代码的静态分析。

2. 有利于发现错误。

```ts
let obj = { message: "" };
console.log(obj.messege);
```

3. 更好的 IDE 支持，做到语法提示和自动补全。
4. 提供了代码文档。
5. 有助于代码重构。

## 5.静态类型的缺点

1. 丧失了动态类型的代码灵活性。 JavaScript 之所以被那么多人喜欢不就是因为用起来没那么多的限制

2. 增加了编程工作量。

3. 更高的学习成本

4. 引入了独立的编译步骤

5. 兼容性问题
