# Hello,World

JavaScript 脚本的引入方式

  • script
<!DOCUMENT HTML>
<html>
    <body>
        <script>
            alert('Hello world');
        </script>
    </body>    
    </html>
  • 外部脚本
<script src='/js/script1.js'></script>

注意： 一般来说，只有简单的脚本才嵌入HTML中，更复杂的使用外部脚本的方式。

使用独立脚本的好处是浏览器下载它，然后会将它保存到浏览器的缓存open in new window中。之后其他页面想用相同的脚本中就会从缓存中获取，而不是下载它。所以文件实际上只会下载一次。这样可以节省流量，并使页面加载更快

如果设置了src属性,script标签内容会被忽略。
